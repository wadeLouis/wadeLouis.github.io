打开端口转发功能

  • 临时设置

sudo echo 1 >/proc/sys/net/ipv4/ip_forward

  • 设置永久生效

vim 打开 /etc/sysctl.conf文件 ，在文件中添加net.ipv4.ip_forward=1，通过命令sudo sysctl -p使其生效
