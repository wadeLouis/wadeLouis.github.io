循环测试

什么意思呢？就是某一测试项重复几次的意思，主要通过repeat这个方法实现。 如何做？ repeat函数的声明格式如下: * repeat(times){chain} times 是次数，chain是每一次的动作 * repeat(times, counterName){chain} times是次数，counterName是保存当前次数的变量名，chain是每一次的动作

具体代码：

object Browse {
// 这里是重复做5次
  val browse = repeat(5, "n") { // 1   
    exec(http("Page ${n}")
      .get("/computers?p=${n}")) // 2
      .pause(1)
  }
}

repeate的chain部分这个块中的代码也可以直接用exec()运行一个函数，当然如果要传参过去，只能传String了，这里的http(“Page ${n}”)就是个方法是不是💪…

响应检查和失败管理

检查响应内容在上节已经提到，具体可以去看上节内容。 失败管理有三种： * tryMax(times){chain} 重试直到成功，如果一直不成功就重复到设定的最大数 * exitBlockOnFail{chain} 如果失败就退出当前执行块，scenario继续执行下面的动作 * exitHereIfFailed scenario直接退出

例子：

val tryMaxEdit = tryMax(2) { // 1 重试2次
  exec(edit)
}.exitHereIfFailed // 2次都不成功就退出整个scenario
val tryExitBlock = {
    exec(...)
    exitBlockOnFail{
      exec(...)
    }
    exec(...) // 这里继续会执行
}
